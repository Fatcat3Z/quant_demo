# stock_trade_demo Docs

本目录用于存放 `stock_trade_demo` 子项目专属的说明文档、补充设计文档与历史文档整理后的归位文件。

## 适合放入本目录的内容

- 子项目专属说明
- 运行说明、设计补充、专题文档
- 原先散落在 `stock_trade_demo/` 根目录但不属于源码/数据/产物的文档

## 暂行规则

1. `stock_trade_demo/README.md` 仍保留在项目根目录作为主入口说明。
2. 其他子项目专属文档后续可逐步整理到本目录。
3. 不要把自动生成报表放到这里；报表应进入 `stock_trade_demo/reports/`。
