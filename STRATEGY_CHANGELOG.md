## v1.0 — 原版策略 (基准)

**文件**: `original_strategy()`  
**日期**: 初始版本

### 策略逻辑

```
三步过滤 + 市值排名:

Step 1: 行业估值分位过滤
  - 按申万二级行业分组，计算 PE倒数/PB倒数 的中位数
  - expanding rank (min 12 periods) → 估值分位 = (EP分位 + BP分位) / 2
  - 过滤: val_pct < 0.68 (剔除高估值行业 TOP 32%)

Step 2: bias_20 过滤
  - 每截面 bias_20 < 52%分位数
  - 剔除股价偏离MA20过高的股票（追高风险）

Step 3: 成交额波动过滤
  - 每截面 成交额std_10 < 78%分位数
  - 排除异常交易/庄股

排名因子: 总市值 (越小越好)
  - 选股数: 6只/月
```

### 关键指标

| 指标 | 数值 |
|------|------|
| 累积净值 | 5,223.19 |
| 年化收益 | 55.51% |
| 最大回撤 | -71.50% (2008-01-31 → 2008-09-26) |
| 年化收益/回撤比 | 0.78 |
| 月度胜率 | 66.24% |
| 牛市月均收益 | +6.05% (184月) |
| 熊市月均收益 | -1.83% (50月) |
| 滚动12月平均最大回撤 | -19.48% |

### 策略特征
- **极端小市值偏好**: 因子是纯市值排名，天然选择市场上最小的股票
- **A股小市值溢价**: 2007-2026年间A股小市值效应极强，累积净值达5000x+
- **高波动**: 最大回撤-71.50%发生在2008年金融危机期间
- **牛熊不对称**: 牛市年化102.4% vs 熊市年化-19.9%

## v1.1 — 缠论增强策略 (代理因子)

**文件**: `chan_enhanced_strategy()`  
**日期**: 2026-05-16 (Task 1.1)  
**因子来源**: `compute_chan_factors()` — 月度频率代理计算

### 变更内容

基于 `quant_factor.md` Sections 10-14，从月度OHLC+MACD数据计算缠论代理因子：

- **分型 (Fractal)**: 3-K线极值检测（当月 vs lag1 vs lag2）
- **背驰 (Divergence)**: MACD bar对比（价格跌+MACD升=底背驰，反之为顶背驰）
- **中枢 (Zhongshu)**: ZG = min(high, high_l1, high_l2), ZD = max(low, low_l1, low_l2)
- **笔 (Stroke)**: 方向 = sign(涨跌幅_20)，强度 = abs(涨跌幅_20)/std
- **买卖点**: 加权综合得分（一类买点3分 + 二类买点2分 + 三类买点1分 - 卖点惩罚）

### 策略修改

```
在 v1.0 三步过滤基础上增加:

Step 4: 缠论负向排除
  - 排除同时满足: 中枢上方 + 顶背驰 + 顶分型（三重确认强卖点）

Step 5: 缠论排名倾斜（3%）
  - rank_final = rank_size * (1.0 - 0.03 * chan_signal_norm)
  - Chan信号最好 → 排名打97折（3%优先）
  - Chan信号最差 → 排名打103折（3%惩罚）
```

| 累积净值 | 5,130.98 | -1.77% |
| 年化收益 | 55.37% | -0.14pp |
| 最大回撤 | -71.57% | -0.07pp |
| 月度胜率 | 66.24% | 持平 |

| 跑赢原版月份 | 1/234 (0.4%) | — |
| 平均月度超额 | -0.01% | — |

### 结论

❌ **缠论代理因子未能超越原版。** 原因:

1. **频率错配**: 缠论是为日线/日内数据设计的，月度3-K线代理丢失了笔→线段→中枢的结构信息
2. **信号方向反转**: 底分型 IC = -0.0118 (t=-2.86)，月度频率下"买低"信号反而与动量方向冲突
3. **市值效应压倒一切**: 3%的Chan倾斜无法克服20年5000x+的小市值溢价
4. **无回撤保护**: 滚动12月回撤-19.49%与原版-19.48%几乎相同

## v2.0 — Method A: 日线缠论流水线 → 月度聚合

**文件**: `method_a_strategy()`  
**日期**: 2026-05-16 (Method A Optimization)  
**因子来源**: `chan_monthly_factor_builder.py` — 对每只股票跑 `ChanTheoryAnalyzer` 完整日线流水线后月度聚合  
**数据管道**: Sina API 500日K线 → InclusionProcessor → FractalDetector → StrokeBuilder → SegmentBuilder → ZhongshuDetector → DivergenceDetector → TradeSignalGenerator → 月度聚合

### 变更内容

**核心改进**: 使用真实日线缠论结构替换月度代理近似：

| 因子维度 | v1.1 (代理) | v2.0 (Method A) |
|---------|-----------|----------------|
| 分型 | 3-K线月度极值检测 | 日线包含处理→分型检测→分型合并 |
| 笔 | sign(涨跌幅_20) 代理 | 日线笔构建（相邻分型确认） |
| 线段 | — | 日线线段构建（笔重叠检测） |
| 中枢 | 3-bar min/max 近似 | 日线中枢检测（线段/笔重叠→ZG/ZD） |
| 背驰 | MACD bar简单对比 | MACD面积对比（笔级背驰检测） |
| 买卖点 | 启发式加权 | 三类买卖点系统检测 |

**输出因子 (16个)**:
`ma_top_fractal`, `ma_bottom_fractal`, `ma_fractal_ratio`, `ma_stroke_dir`, `ma_stroke_count`, `ma_stroke_strength`, `ma_zhongshu_count`, `ma_zhongshu_position`, `ma_zhongshu_width`, `ma_top_div`, `ma_bottom_div`, `ma_div_signal`, `ma_buy_signals`, `ma_sell_signals`, `ma_segment_count`

### 策略逻辑

```
遵循 PM 建议: "在小市值桶内用缠论因子排序"

Step 1-3: 与原版相同的三步过滤（行业估值 + bias_20 + 成交额波动）

Step 4: Method A 缠论负向排除
  - 排除同时满足: 顶背驰(ma_div_signal=-1) + 中枢上方(ma_zhongshu_position>0.5) + 卖>买

Step 5: Method A 综合得分
  ma_score = +1.0*bottom_fractal -1.0*top_fractal
             +3.0*bottom_div -3.0*top_div
             +2.0*buy_signals -2.0*sell_signals
             +1.5*(below_zs) -1.5*(above_zs)

Step 6: 排名 = 市值排名 * (1.0 - 0.05 * ma_score_norm)
  - 5% Chan倾斜（vs v1.1的3%）
  - Chan因子仅在市值接近时起tiebreaker作用
```

### 当前局限

⚠️ **Method A与v1.0表现完全相同的原因**:

1. **因子覆盖稀疏**: 200只股票的Method A因子仅覆盖3.7%的选股池
2. **5%倾斜无效**: 在97%+的股票没有Method A因子时（中性填充），策略退化回纯市值排名
3. **Chan因子IC方向**: 背驰强度 IC = -0.0217 (t=-3.46)，底分型 IC = -0.0118 (t=-2.86)，均指向负向——增加Chan权重可能反而伤害收益

### 后续优化方向

1. **扩大因子覆盖**: 对1000+只股票跑Method A流水线（需要更多API调用时间）
2. **分桶策略**: 先按市值分5桶，仅在最小组内用Chan因子排序，其他桶保持纯市值
3. **IC导向权重**: 基于因子IC测试结果，对有效因子（中枢下方IC=+0.0037）给正权，无效因子（底分型IC=-0.0118）反向使用
4. **日线级别交易**: 缠论信号应在更短持仓周期（周度/日度）而非月度使用

## 总体结论

1. **A股小市值效应是压倒性的alpha来源** — 任何偏离纯市值排名的因子增强都会稀释这一效应
2. **缠论因子在月度频率下信号方向与直觉相反** — 底分型/底背驰的IC显著为负，需要反向使用或更换频率
3. **Method A (日线流水线) 在理论上优于代理因子** — 真实笔/线段/中枢结构更准确，但需要更大的覆盖面和更合理的权重设计
4. **缠论最适合的应用场景是风险管理和入场时机** — 而非横截面排序因子。建议用缠论信号作为持仓止损/止盈的触发条件，而非选股排名
